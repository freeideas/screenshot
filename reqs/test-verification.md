# File Output Requirements

**Source:** ./README.md

Requirements for file creation and output handling when saving screenshots.

## $REQ_OUTPUT_001: Create Output File at Specified Path
**Source:** ./README.md (Section: "Usage")

screenshot.exe must create a PNG file at the specified output path.

## $REQ_OUTPUT_002: Output PNG Format
**Source:** ./README.md (Section: "Technical Details")

The output file must be in PNG format.

## $REQ_OUTPUT_003: Include Window Decorations
**Source:** ./README.md (Section: "Technical Details")

The captured screenshot must include the full window including title bar and decorations.

## $REQ_OUTPUT_004: Output File Has Content
**Source:** ./readme/TESTING.md (Section: "What We Test")

The output PNG file must have a size greater than 0 bytes.
